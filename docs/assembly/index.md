# assembly-cli

`assembly-cli` provides governance visibility and automation for Assembly contracts on Abstract.

## Install

```bash
pnpm add -g @spectratools/assembly-cli
```

## Focus areas

- member lifecycle and registry state
- council seats and auction slots
- forum threads/comments/petitions
- governance proposals and vote participation
- treasury balances and spend controls
