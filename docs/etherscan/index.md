# etherscan-cli

`etherscan-cli` is a multi-chain Etherscan V2 API client for explorer-centric automation.

## Install

```bash
pnpm add -g @spectratools/etherscan-cli
```

## Focus areas

- account balances and transfer history
- contract ABI/source/deployment metadata
- transaction info, receipts, and status
- token metadata, holder snapshots, and supply
- gas oracle and confirmation estimates
