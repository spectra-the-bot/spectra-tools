# erc8004-cli

`erc8004-cli` supports trustless-agent identity, reputation, validation, and discovery operations on Abstract.

## Install

```bash
pnpm add -g @spectratools/erc8004-cli
```

## Focus areas

- identity registry reads and writes
- registration file fetch/validate/generate
- reputation scoring and feedback history
- validator request/status/result flows
- metadata search and canonical resolver lookups
