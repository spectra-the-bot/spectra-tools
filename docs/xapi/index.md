# xapi-cli

`xapi-cli` wraps X (Twitter) API v2 endpoints for feed intelligence, content operations, and messaging workflows.

## Install

```bash
pnpm add -g @spectratools/xapi-cli
```

## Focus areas

- post lookup, search, create, and delete
- user profile and social graph queries
- home/mentions timeline polling
- list and trend discovery
- direct message send/read flows
