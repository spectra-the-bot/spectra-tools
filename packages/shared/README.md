# @spectra-the-bot/cli-shared

Shared middleware, utilities, and testing helpers for spectra CLI tools.

## Installation

```bash
pnpm add @spectra-the-bot/cli-shared
```

## Exports

- `middleware` — `apiKeyAuth`, `withRetry`, `createRateLimiter`, `paginateCursor`, `paginateOffset`
- `utils` — `createHttpClient`, `weiToEth`, `checksumAddress`, `formatTimestamp`, `truncate`
- `testing` — `createMockServer`
